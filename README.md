# Filer-rs
Simple File Manager
Controls <br>
C = Copy <br>
P = Paste <br>
UP/Down/Left arrow keys = navigation <br>
D = Delete <br>
Q = Quit Program <br>
<br>
Note: is in experimental state & isnt meant to be used for daily activities.
<br>
To-DO List <br>
File Extension icons <br>
Viewport for text & pdf & images <br>
Built in vim like text editor <br>
